# pm2-notify
PM2 Log error email notifier

### Installation
```sh
$ npm ci
# Edit config
$ vim src/config.ts
```

### Start
```sh
$ npm start
# OR
$ npm run build
$ npm run pm2:start
```

### Using
* [MJML](https://mjml.io) for Email Markup
* [Handlebars](https://handlebarsjs.com/) for Template Parser
